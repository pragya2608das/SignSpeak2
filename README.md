# SignSpeak 2.0 🤟

AI-powered Sign Language Translation System.

## Features

- CTR-GCN Sign Recognition
- WLASL 1000 Dataset
- Gemini AI Sentence Translation
- Text-to-Speech Output
- Video Upload Prediction
- Live Webcam Mode (In Progress)

## Tech Stack

- Python
- Streamlit
- MediaPipe
- CTR-GCN
- Gemini 2.5 Flash
- PyTorch
